Правила игры:

  1. Для начала игры "Угадай Число" нажмите кнопку "Сыграть в игру"
  2. Выберите число, которое задумали из диапазона от 1 до 100
  3. Напишите выбранное число в поле ввода
  4. Кнопка "Угадать" - нужна для проверки выбранного вами числа с загаданным
  5. Меняй съчисло следуя простой логике подсказок
